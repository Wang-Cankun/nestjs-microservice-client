
# nestjs-microservice-client

## Description

This is an example repo for:

- [Nest](https://github.com/nestjs/nest) client side [microservices](https://docs.nestjs.com/microservices/basics)


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
